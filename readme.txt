== Description ==

> Did you find this plugin helpful? Please consider [leaving a 5-star review](https://wordpress.org/support/plugin/creatus-extended/reviews/).

Creatus Extended adds demos installation utility, page builder template library, widgets, widgets ID and class, editor formats and additional functionalities to Creatus WordPress Theme.

This plugin runs only if the [Creatus](https://github.com/Themezly/Creatus) theme is installed and active.

[vimeo https://vimeo.com/302178559]

**Get involved**
Developers can contribute to the source code on the [Creatus Extended GitHub Repository](https://github.com/Themezly/Creatus-Extended).

= Minimum Requirements =

* WordPress 4.0 or greater
* PHP version 5.6 or greater

= Installation =

1. Upload `Creatus-Extended-version_number.zip` to the `/wp-content/plugins/` directory and extract.
2. Activate the plugin by going to **Plugins** page in WordPress admin and clicking on **Activate** link.
